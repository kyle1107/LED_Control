# LED_Control

LED 제어 프로젝트

기존의 내 방의 전등의 밝기가 너무 밝으며, 침대에 누워서 제어할 수 없는 문제가 발생

이를 해결하기 위하여 무드등 비슷하게 만들어보기로 결심하였다.

빛을 위로 보내어 천장에 반사하여 방 밝기를 높이는 방법으로 제작
